# 🔥 Rap Density Portal

AI-powered rap lyric analysis tool.

## Features
- Rhyme density scoring
- Multisyllable detection
- Flow analysis
- Vocabulary tracking
- AI-style verse feedback

## Tech Stack
- Python (Flask)
- HTML / CSS
- Pronouncing library

## Purpose
Built to analyze lyrical complexity, rhyme structure, and performance quality of rap verses in real time.
